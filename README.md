# PROG102_BIT1201F_GROUP8
WorkLink SL

Project Overview

WorkLink SL is a Digital Public Good (DPG) designed to connect job seekers and employers across Sierra Leone through a modern digital recruitment platform. The system provides an efficient and user-friendly environment where job seekers can search and apply for job opportunities, while employers can post vacancies, manage applications, and recruit qualified candidates.

The project supports United Nations Sustainable Development Goal (SDG) 8: Decent Work and Economic Growth by promoting employment accessibility and improving workforce connectivity. Through the use of digital technology, WorkLink SL helps bridge the gap between available talent and employment opportunities, contributing to economic development and professional growth.

Built using software engineering best practices, the platform follows a structured Software Development Life Cycle (SDLC) and utilizes a 3-Tier Architecture consisting of the Presentation Layer, Application Layer, and Database Layer. The system is designed to be scalable, secure, and maintainable.

Key Features

* User Registration and Authentication
* Job Search and Filtering
* Job Application Management
* Employer Job Posting System
* Interview Scheduling and Candidate Management
* User Messaging and Notifications
* Administrative Monitoring and User Management
* GitHub-Based Version Control and Documentation

Technologies Used

* Figma (UI/UX Design)
* Git & GitHub (Version Control)
* UML Modeling (Use Case, Activity, and Class Diagrams)
* Relational Database Design
* Software Development Life Cycle (SDLC)

Project Goal

The primary goal of WorkLink SL is to create a reliable and accessible job recruitment ecosystem that simplifies the hiring process, empowers job seekers, supports employers, and contributes to sustainable economic growth in Sierra Leone.
